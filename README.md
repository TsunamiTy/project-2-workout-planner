# Workout-planner
An application that allows you to sign up and log into your personal account and view, edit, add, and delete exercises.

## Technologies Used
- HTML
- JavaScript
- CSS
- Node.js
- Express
- EJS
- MongoDB
- Mongoose
- Google Fonts
- Imgur
